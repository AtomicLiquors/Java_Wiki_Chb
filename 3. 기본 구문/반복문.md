- do - while
    
    조건검사 없이 일단 한 번은 수행한다.
    
    ```jsx
    package test01;
    
    public class WhileTest {
    
    	public static void main(String[] args) {
    		// TODO Auto-generated method stub
    		
    		int i = 0;
    		int result=0;
    		do {
    			result += i;
    			System.out.printf("i= %d, result = %d\n", i, result);
    			i++;
    		}while(i<10);
    		System.out.println("i="+i);
    
    	}
    
    }
    ```
