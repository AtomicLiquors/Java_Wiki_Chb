# 정규식
조건에 맞는 데이터 추출  


텍스트 데이터 중에서 원하는 패턴과 일치하는 문자열을 찾아내기 위해,
미리 정의된 기호와 문자로 작성한 문자열.  

원래 Unix와 Perl에서 사용하였다.  
제대로 배우려면 책 한권을 써야되는 내용이니,  
자주 사용하는 정규식 위주로 응용하는 법만 배우자.

&nbsp;  
### 활용방법
- 방대한 텍스트에서 원하는 데이터를 추출한다.
- 입력 데이터가 형식에 맞는지 확인한다.
